Hi everyone! 👋

I'm new to APKM package development and I'd like to create my first package. I've read the documentation but I'm still a bit confused about the process.

Here's what I've understood so far:

  • I need to create a PKGBUILD file
  • The package should follow a specific directory structure
  • I need to use BOOL to build the package

But I have some questions:

  1. What's the minimal PKGBUILD for a simple C program?
  2. How do I test my package locally before uploading?
  3. What's the difference between release (r0, r1) and version?

Here's my current PKGBUILD attempt:

pkgname=hello-world
pkgver=1.0.0
pkgrel=r0
pkgdesc="A simple hello world program"
arch=('x86_64')
license=('MIT')

build() {
    gcc -o hello hello.c
}

package() {
    install -Dm755 hello "$pkgdir/usr/bin/hello"
}

Is this correct? Any help would be greatly appreciated! 🙏

Hey John! Your PKGBUILD looks good for a start! Here are some suggestions:

1. Add a `source` array to specify where your source code comes from:

source=("hello-world-$pkgver.tar.gz")
sha256sums=('SKIP')

2. For local testing, you can use `bool --verify` to check your package:

bool --verify hello-world-1.0.0-r0-x86_64.tar.bool

3. Release (r0, r1) is for rebuilds without version change (like fixing a build issue), while version changes when the software itself updates.

Hope this helps! 😊

Great question! Also make sure to:

  • Add a `license` array with the appropriate license
  • Include a README.md file in your package
  • Test installation with `apkm install ./hello-world-1.0.0-r0-x86_64.tar.bool`

Here's an enhanced PKGBUILD:

pkgname=hello-world
pkgver=1.0.0
pkgrel=r0
pkgdesc="A simple hello world program"
arch=('x86_64')
url="https://github.com/johndoe/hello-world"
license=('MIT')
depends=('glibc')
source=("$pkgname-$pkgver.tar.gz::https://github.com/johndoe/hello-world/archive/v$pkgver.tar.gz")
sha256sums=('SKIP')

build() {
    cd "$srcdir/$pkgname-$pkgver"
    gcc -o hello hello.c
}

package() {
    cd "$srcdir/$pkgname-$pkgver"
    install -Dm755 hello "$pkgdir/usr/bin/hello"
    install -Dm644 README.md "$pkgdir/usr/share/doc/$pkgname/README.md"
}
