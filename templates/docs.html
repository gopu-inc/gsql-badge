Getting Started

Welcome to the Zarch ecosystem! Install GPM and start using Goscript packages in minutes.

Installation

shell curl -sL https://zenv-hub.onrender.com/install.sh | sh

This installs GPM (Goscript Package Manager) and all necessary tools.

shell gpm --version
gpm doctor
💡 Tip: GPM stores config in ~/.gpmrc. Packages go to /usr/local/lib/goscript/.

GPM Basics

Initialize a Project

shellgpm init my-project

Install Packages

shellgpm install http # Latest version
gpm install http@1.2.0 # Specific version
gpm install # All from Manifest.toml

Commands

  • gpm install <pkg> - Install a package
  • gpm uninstall <pkg> - Remove a package
  • gpm update <pkg> - Update to latest
  • gpm search <query> - Search registry
  • gpm publish - Publish your package
  • gpm build - Build .tar.bool archive
  • gpm login <user> <pass> - Authenticate
  • gpm run <script> - Execute a script

Manifest.toml

toml [metadata]
name = "my-package"
version = "1.0.0"
release = "r0"
arch = "x86_64"
description = "My awesome package"
author = "username"
license = "MIT"
main = "main.gjs"

[dependencies]
http = "^1.2.0"
fs = "*"

[dev-dependencies]
testlib = "~0.5"

[scripts]
build = "gpm build"
test = "gpm run test.gjs"

Goscript Language

Goscript is a modern scripting language with async support, FFI, and powerful pattern matching.

Variables

goscriptlt name = "Goscript" # Immutable
muts counter = 0 # Mutable
cn PI = 3.14159 # Constant

Functions

goscriptfn add(a: int, b: int): int { ret a + b }

async fn fetch(url: string) { ret await f"curl -s '{url}'" }

Structs & Enums

goscriptstruct User { name: string, age: int }
enm Status { Active, Inactive }

Async/Await

goscriptasync fn fetch_data(url: string) {
try {
ret await f"curl -s '{url}'"
} catch (e) {
println(f"Error: {e}")
ret ""
}
}

Publishing Packages

⚠️ Important: Ensure your Manifest.toml has valid name, version, and arch.
shellgpm login username password
gpm build
gpm publish

Dependencies Management

Dependencies are declared in Manifest.toml under [dependencies] and [dev-dependencies].

GPM resolves the full dependency tree, hoists common packages, and generates a gpm.lock file.

toml[dependencies]
http = "^1.0" # Compatible with >=1.0.0 <2.0.0
fs = "~2.5" # Compatible with >=2.5.0 <2.6.0
json = "*" # Any version
