Getting Started

Install Zarch Hub tools in one command:

curl -sSL https://gsql-badge.onrender.com/install.sh | sudo bash

Or install from source:

git clone https://github.com/gopu-inc/apkm-gest
cd apkm-gest
mkdir build && cd build
cmake ..
make -j4
sudo make install

APKM - Package Manager

Basic commands:

# Install a package
apkm install nginx

# Search for packages
apkm search database

# Show package info
apkm info python

# List installed packages
apkm list

# Update repository databases
apkm update

# Upgrade all packages
apkm upgrade

APSM - Publisher

Publish packages to Zarch Hub:

# Login to Zarch Hub
apsm login

# Push a package (automatically reads Manifest.toml)
apsm push mypackage-v1.0.0-r1.x86_64.tar.bool

# Check authentication status
apsm status

# Logout
apsm logout

BOOL - Package Builder

Build packages from APKMBUILD:

# Initialize a new project
bool --init

# Build package (creates .tar.bool)
bool --build

# Show package info
bool --info mypackage-v1.0.0-r1.x86_64.tar.bool

# Verify package integrity
bool --verify mypackage-v1.0.0-r1.x86_64.tar.bool

GET - Manifest Parser

Parse and display Manifest.toml files:

# Display entire manifest
get Manifest.toml

# Display specific section
get Manifest.toml metadata

# Display with colors
get --color Manifest.toml

Manifest.toml Format

Example manifest:

[metadata]
name = "myapp"
version = "1.0.0"
release = "r1"
arch = "x86_64"
description = "My awesome application"
maintainer = "Your Name "
license = "MIT"

[metadata.badges]
version = { label = "version", color = "blue" }
license = { label = "license", color = "yellow" }

[dependencies]
libc = ">=2.30"
gcc = ">=10.0"

[build-dependencies]
cmake = ">=3.20"
