Qu'est-ce que GSQL Badges ?

Utilisation de l'API

Créez des badges programmatiquement :

// Requête POST vers /create
curl -X POST https://gsql-badge.onrender.com/create \\
  -H "Content-Type: application/json" \\
  -d '{{
    "slug": "python",
    "label": "Python",
    "message": "3.12",
    "color": "#3776ab",
    "logo_url": "https://www.python.org/static/favicon.ico"
  }}'

Endpoints disponibles :

  • GET /badge - Badge officiel
  • GET /badge/<slug> - Récupérer un badge
  • POST /create - Créer/modifier un badge (JSON)
  • POST /upload - Upload via formulaire
  • GET /list - Lister tous les badges
  • GET /health - Vérifier l'état du service

message string Texte de droite (max 100 chars) 1.0.0
color string Couleur hexadécimale #22c55e
logo_url string URL du logo à télécharger https://.../logo.png
